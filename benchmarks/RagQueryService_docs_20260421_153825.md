#### Method: QueryAsync

Executes the full RAG query pipeline.

1.  **Embedding:** Converts the input `request.Query` into a vector embedding using `_embeddingService`.
2.  **Retrieval:** Iterates through all `request.Collections`, performing a semantic search against Weaviate for each collection using the query embedding.
3.  **Sorting & Filtering:** Combines results from all collections and sorts them based on the proximity distance (closer to 0 is better), taking the top `request.MaxResults`.
4.  **Source Preparation:** If `IncludeSources` is true, it maps the retrieved results into `SourceDocument` objects, calculating a score and parsing metadata.
5.  **Context Building:** Calls `BuildContext` to format the retrieved content into a single, structured context string, separating information by knowledge base collection (e.g., "=== CUSTOMER INFORMATION ===").
6.  **LLM Prompting:** Constructs a detailed `OllamaChatRequest`. The system message contains a fixed `SystemPromptTemplate` which strictly instructs the LLM on its role, constraints (answer ONLY based on context), and required output format. The user message is the original query.
7.  **Generation:** Sends the structured chat request to `_ollamaClient.ChatAsync`.
8.  **Response Handling:** Extracts the generated answer and records the total query time.
9.  **Error Handling:** Includes robust `try-catch` logic to log errors and return a generic failure message if any step fails.
