### Method: QueryAsync

The primary execution method for performing a RAG query. It follows these steps:
1. Measures the execution time.
2. Generates a vector embedding for the user's query.
3. Iterates through all specified collections, performing a semantic search in Weaviate for each.
4. Aggregates and sorts the results based on the distance score (closest first).
5. If requested, formats the retrieved documents into `SourceDocument` objects.
6. Constructs a detailed context string by grouping results based on predefined collections (Customers, WebsiteNavigation, etc.).
7. Formats the system prompt using a fixed template, injecting the context and instructions.
8. Calls the `OllamaClient` to generate the final answer using the system prompt and the user query.
9. Logs the completion time and returns the structured response.
