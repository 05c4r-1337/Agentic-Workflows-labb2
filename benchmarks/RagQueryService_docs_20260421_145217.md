## Interface: IRagQueryService

Defines the contract for services responsible for executing Retrieval-Augmented Generation (RAG) queries. This service takes a user request, retrieves relevant context from external knowledge bases (like Weaviate), and then uses a Large Language Model (LLM) via Ollama to generate a final, grounded answer.
