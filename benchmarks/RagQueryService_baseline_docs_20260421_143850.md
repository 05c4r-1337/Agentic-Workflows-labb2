### Interface: IRagQueryService

Defines the contract for executing a Retrieval-Augmented Generation (RAG) query. This service handles the entire process from embedding the user query to retrieving context and generating a final answer using an LLM.

#### Method: QueryAsync

Executes the RAG process. It takes a user query, retrieves relevant documents from specified knowledge bases (Weaviate), formats these documents into a context, and then uses an LLM (Ollama) to generate a final answer based solely on that context.

| Parameter | Type | Description |
| :--- | :--- | :--- |

| Return Type | Description |
| :--- | :--- |
| `Task<RagQueryResponse>` | A task that resolves to a `RagQueryResponse` object containing the generated answer, any retrieved sources, and the total query processing time. |

**Example Usage:**

```csharp
// Assume services are injected and configured
var ragService = serviceProvider.GetRequiredService<IRagQueryService>();

// 1. Prepare the request
var request = new RagQueryRequest

    Collections = new List<string> { "WebsiteNavigation", "Customers" },

    IncludeSources = true
```
