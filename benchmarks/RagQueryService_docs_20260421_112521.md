#### Method QueryAsync

Executes the full RAG query pipeline. This involves:
1. Generating an embedding for the user query using `IEmbeddingService`.
2. Performing semantic searches across all specified collections in Weaviate (`IWeaviateService`).
3. Sorting and selecting the top results based on the distance score.
4. Building a structured context string from the retrieved documents using `BuildContext`.
5. Calling the LLM (Ollama) with a system prompt containing the context and the user query.
6. Calculating and logging the total execution time.

**Algorithmic Detail (Scoring):**
The method calculates the source score for the final response by transforming the raw distance metric (`D`) retrieved from Weaviate into a similarity score (`S`) using the formula: $S = 1 - D$. This score is used in the `SourceDocument` object.
