### QueryAsync(RagQueryRequest request)

(See `IRagQueryService` documentation for detailed parameter and return value descriptions.)

This method executes the full RAG pipeline:
1. Calculates the embedding for the input query.
2. Iterates through all specified collections, performing a semantic search against Weaviate.
3. Aggregates and sorts the results. The sorting is performed by ordering results based on the `Distance` metric provided in the search result's additional data. Results where the distance is null or missing are treated as having the highest possible distance (`double.MaxValue`) and are thus placed at the end of the sorted list.
4. Constructs the context string by grouping results into specific sections (Customers, WebsiteNavigation, Additional).
5. Formats the system prompt using the predefined template, injecting the compiled context. The system prompt strictly instructs the LLM to answer *only* based on the provided context and specifies structured roles for different types of information (e.g., customer details, navigation steps).
6. Calls the `OllamaClient` to generate the final answer based on the system prompt and user query.
7. Measures and records the total query time. If an exception occurs at any point, the function catches the error, logs it, and returns a standardized error response.
