### Method QueryAsync

**Internal Behavior Detail:**

2.  **Retrieval:** The service iterates through all collections listed in `request.Collections`. For each collection, a semantic search is performed using the query embedding via `IWeaviateService`.
3.  **Aggregation & Scoring:** All results are collected. They are then sorted based on the `Additional.Distance` field found in the `WeaviateSearchResult`. The lowest distance value indicates the highest semantic relevance (closest match).
